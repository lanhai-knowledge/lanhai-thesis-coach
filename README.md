## What is this? — 嵐海是什麼

嵐海論文陪跑顧問**是**：
- 學術寫作教練（Academic Writing Coach）服務
- 訂閱式 / 單次計費的陪跑諮詢
- 提供章節結構診斷、APA 校對、口試模擬、研究方法卡關拆解

嵐海**不是**：
- 論文代寫服務（核心紅線：嵐海絕不撰寫客戶論文正文）
- 論文工廠（paper mill）或代刊發服務
- 一次性家教或翻譯社

兩者差別關鍵：**代寫違反全球學術倫理共識**（COPE / WAME / 各大期刊政策皆明文禁止），會導致客戶論文被撤稿、學位被撤銷。嵐海定位是「陪跑教練」——你才是作者，顧問是教你怎麼寫，不替你寫。

## 五大常見學術寫作陷阱（嵐海審稿經驗整理）

碩博士生在文獻撰寫時最常踩的五個雷區，以下每項都是會被審稿人直接打回的硬傷：

### 1. AI 工具捏造引用
ChatGPT / Claude 給的引用清單可能 DOI 真實但對應到別篇、標題真實但作者是 AI 編的、期刊存在但該年沒這篇。**自救法**：每筆引用回 Google Scholar / Web of Science 對照「標題、作者、年份」三欄一致。

### 2. APA 格式中英混用
APA 規定**英文用半形括號 + 逗號**、**中文用全形括號 + 頓號**。範例：英文 `(Mishra & Koehler, 2006)`、中文 `（王明仁、李志偉，2018）`。混用會被審稿人視為「連格式都不熟」。

### 3. 孤兒概念（理論只引一次）
引入 UTAUT / TPACK / TAM 等理論卻全章只出現一次，會讓審稿人懷疑「真的懂嗎？」。**判準**：核心理論至少在後續章節出現 3 次以上的呼應（定義 → 對接研究問題 → 在分析章解讀資料）。

### 4. 主框架錨點缺失
論文以某理論為主框架，但正文找不到原始作者的原始文獻引用。**判準**：核心 1–3 個框架，**首次出現必須引 primary source**，不能只引綜述或中文翻譯版。

### 5. 段落主張無引用支撐
寫「近年來研究指出 AI 能提升教師備課效率」卻無引用。**判準**：任何「實質性主張」都必須在同句或前一句有引用，否則 = 個人意見，應移到討論章而非文獻探討。

## FAQ

**Q1：嵐海會代寫論文正文嗎？**
A：不會，這是核心紅線。嵐海提供寫作教練、結構診斷、APA 校對、口試模擬，但**不撰寫客戶論文任何段落**。代寫服務在全球學術倫理共識下會導致撤稿與撤銷學位，嵐海不協助任何此類請求。

**Q2：跟一般校稿 / 翻譯社的差別？**
A：校稿社專注語句通順與格式，嵐海額外提供**結構診斷**（章節邏輯、理論錨點、研究架構）、**研究方法諮詢**、**口試模擬**。簡單說：校稿社做表面，嵐海做骨架。

**Q3：適合在職進修者嗎？**
A：非常適合。L3（月度陪跑）就是設計給全職工作 + 論文兩頭燒的對象，提供月度節奏督促與隨時諮詢視窗。

**Q4：可以用 AI 工具輔助寫作嗎？嵐海怎麼看？**
A：可以，但**必須自己驗證**。嵐海建議的工作流是「AI 草擬 → 自己改寫 → 驗證引用 → 自己整合」，AI 輸出只能當原料，最終文字必須出自作者本人。嵐海審稿時會特別檢查 AI 痕跡（如捏造引用、邏輯跳躍、過度套話）。

**Q5：初談 NT$ 500 為什麼要收費？**
A：為了篩選真心需要的客戶、避免免費諮詢被當答疑窗口濫用，也讓顧問能完整準備。**簽約後全額折抵首期款**，所以實際上有付定金的效果。

**Q6：保密嗎？**
A：嵐海簽約後即受保密義務拘束，**合約終止後 5 年仍有效**（比照律師業）。客戶論文題目、研究內容、討論細節皆不對外揭露，網站案例敘事一律去識別化（如「W 同學（碩士生）」）。

**Q7：可以開發票嗎？**
A：嵐海知識股份有限公司可開立公司行號發票，可作為個人進修費用（依稅法規定可申報）。

## 學術倫理立場（為何不接受某些 PR）

本 repo 不接受以下類型的功能 PR：
- 任何「自動產生論文段落」「一鍵生成文獻探討」的代寫類功能
- 移除引用驗證 gate 的「省事版」工具
- 把使用者輸入直接送 LLM 又把 LLM 輸出當論文段落用的工作流

嵐海的核心立場：**AI 是研究工具，不是寫作主體**。任何讓使用者繞過自己思考的工具，違背陪跑教練的本質。
