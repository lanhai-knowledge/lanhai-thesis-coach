五大常見學術寫作陷阱

嵐海審稿經驗整理，碩博士生在文獻撰寫時最常踩的五個雷區。每項都是會被審稿人直接打回的硬傷。

1. AI 工具捏造引用

ChatGPT / Claude 給的引用清單可能 DOI 真實但對應到別篇、標題真實但作者是 AI 編的、期刊存在但該年沒這篇。

自救法：每筆引用回 Google Scholar / Web of Science 對照「標題、作者、年份」三欄一致。

2. APA 格式中英混用

APA 規定英文用半形括號 + 逗號、中文用全形括號 + 頓號。範例：英文 (Mishra & Koehler, 2006)、中文 （王明仁、李志偉，2018）。混用會被審稿人視為「連格式都不熟」。

3. 孤兒概念（理論只引一次）

引入 UTAUT / TPACK / TAM 等理論卻全章只出現一次，會讓審稿人懷疑「真的懂嗎？」。

判準：核心理論至少在後續章節出現 3 次以上的呼應（定義 → 對接研究問題 → 在分析章解讀資料）。

4. 主框架錨點缺失

論文以某理論為主框架，但正文找不到原始作者的原始文獻引用。

判準：核心 1–3 個框架，首次出現必須引 primary source，不能只引綜述或中文翻譯版。

5. 段落主張無引用支撐

寫「近年來研究指出 AI 能提升教師備課效率」卻無引用。

判準：任何「實質性主張」都必須在同句或前一句有引用，否則 = 個人意見，應移到討論章而非文獻探討。

常見問題

Q1：嵐海會代寫論文正文嗎？

不會，這是核心紅線。嵐海提供寫作教練、結構診斷、APA 校對、口試模擬，但不撰寫客戶論文任何段落。代寫服務在全球學術倫理共識下會導致撤稿與撤銷學位，嵐海不協助任何此類請求。

Q2：跟一般校稿 / 翻譯社的差別？

校稿社專注語句通順與格式，嵐海額外提供結構診斷（章節邏輯、理論錨點、研究架構）、研究方法諮詢、口試模擬。簡單說：校稿社做表面，嵐海做骨架。

Q3：適合在職進修者嗎？

非常適合。L3（月度陪跑）就是設計給全職工作 + 論文兩頭燒的對象，提供月度節奏督促與隨時諮詢視窗。

Q4：可以用 AI 工具輔助寫作嗎？

可以，但必須自己驗證。嵐海建議的工作流是「AI 草擬 → 自己改寫 → 驗證引用 → 自己整合」，AI 輸出只能當原料，最終文字必須出自作者本人。嵐海審稿時會特別檢查 AI 痕跡（如捏造引用、邏輯跳躍、過度套話）。
